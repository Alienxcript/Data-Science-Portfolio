# 🩺 Diabetes Data Analysis (NIDDK)

This project presents an exploratory data analysis (EDA) of the Diabetes dataset
provided by the National Institute of Diabetes and Digestive and Kidney Diseases (NIDDK).

## 📌 Objective
To analyze diagnostic health measurements and identify patterns associated
with diabetes occurrence.

## 📊 Dataset
- Source: National Institute of Diabetes and Digestive and Kidney Diseases
- All patients are female and at least 21 years old
- Target variable: `Outcome` (1 = Diabetes, 0 = No Diabetes)

## 🔍 Analysis Steps
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Feature relationships and correlations
- Insight generation

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🚀 Future Work
- Machine learning models for prediction
- Feature engineering
- Model evaluation and explainability
